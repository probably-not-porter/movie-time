How does this work?

This tool provides an easy to configure spinning wheel for Movie Night.
In order to use it, you need a public Google Sheet to keep track of your movies.
You can use this tool for yourself, or modify or change anything for personal use.
The source code can be found here.

Google Sheet ID?

After you create your Google Sheet using the format below, you can paste in the "ID" number from the Sheet.
The ID is a long string of random characters that identifies your Sheet (Don't worry, info is only saved on your browser).
If you are unsure which part of the URL is the ID, it's probably between the /spreadsheets/d/ and /edit in your browser.

Format Guide

You can have as many columns as you want in your spreadsheet.
The only requirement is that is has the following: Title, Type, Suggester, and Watched.
(This is case sensitive)
The Watched column is true/false, and can be replaced by a checkbox or anything else that returns true/false values.
Type and Suggester are case sensitive for values too, and can be used to sort movies on the wheel.

EXAMPLE:
Title Type Suggester Watched
Spongebob Show Alice true
Monsters Inc Movie Bob false
